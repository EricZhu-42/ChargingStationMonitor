# 南哪充电

南京大学 (Nanjing Univerisity) 校内充电站状态监测系统（仅前端部分）

访问地址：https://charge.zhuxh.net/

## 功能介绍

显示南京大学仙林校区内所有电动车充电站的占用状态，约 3~5 分钟完整更新一次。

包含 “已占用时间” “已空闲时间” 等细节信息。
